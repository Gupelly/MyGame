# MyGame
Игра будет представлять из себя метроидванию с видом сбоку. В этом жанре игроку необходимо исследовать мир, но изначально открыты далеко не все локации. 
Получая новые способности или находя некоторые "ключи", герой получает доступ к тем локациям, которые до этого пришлось пропустить. 
По сути этот процесс продолжается до тех пор, пока игра не будет пройдена. 

Сюжет:
Средневековый сеттинг игры. ГГ в качестве наказания за преступление, которое он не совершал, был брошен в лес с монстрами, из которого не возвращаются живыми.	
Игрок должен помочь герою выбраться из леса.

MVP:
1.Несколько экранов с монстрами и платформингом. 
2.Изначально герою доступна хотьба, прыжок, удар мечом. По мере игры герой получает новые способности (двойной прыжок, рывок)
3.Несколько видов монстров и возможно ловушек.
